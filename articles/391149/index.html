New bug in the implementation of POP3 in Outlook 2016 deletes letters

Microsoft recognized a bug in Outlook 2016 latest version 16.0.6568.2025. If the program receives letters using the POP3 protocol and the settings indicate ‚ÄúDelete from server in X days‚Äù, then the letters are deleted from the server immediately.

On the other hand, if this option is disabled, the letters are duplicated.



Microsoft is still thinking about solving the problem, and as a workaround it recommends rolling back to the previous version. To do this, disable automatic updates in File->Office Account->Update Options , and then run the command from the command line

C:\Program Files\Common Files\Microsoft shared\ClickToRun\officec2rclient.exe" /update user updatetoversion=16.0.6366.2068

Another option is to use IMAP instead of POP3.
