The WhatsApp web version is launched.

Officially launched the web version of the popular mobile messenger WhatsApp, which has about 600 million users.

To enter the network, it is required on the service page to scan a QR code, the reader for which is available in the latest version of the application in the menu item WhatsApp Web. This is done in order to observe the principle of linking the program to the user's phone number - in other words, in order to use the web version, you will not need to create and enter a login and password.



The updated instant messenger (with which you can enter the web version) is available only for Android, Windows Phone and BlackBerry. There is no iOS version due to "... Apple platform limitations."

More than a month ago, in the Danish blog Android World , having studied the apk-file of the beta version of WhatsApp, it was suggested that its web version is being prepared for release. As you can see, the guesswork of journalists was correct.
