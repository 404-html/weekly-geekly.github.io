Intel Deep Learning Training Tool Interface

Intel itself presents an equally interesting new free product - Intel Deep Learning SDK , a set of tools for developing, training and deploying software related to a deep study of data. The kit includes: Intel Deep Learning Training Tool, which provides a convenient graphical environment for installing, configuring, and implementing deep learning algorithms, as well as the Intel Deep Learning Deployment Tool, which improves the performance of learning algorithms on Intel platforms. Functions Training Tool:

  • Creating a training dataset
  • Setting up a model with automatically optimized hyper-parameters
  • Launch and monitor various candidate models
  • Visualization of performance and accuracy of training

Deployment Tool features:

  • Import of trained models from all popular deep learning frameworks, regardless of the training hardware platform
  • Compress models to improve performance, reduce space and increase transmission speed
  • Generation of estimator code dependent on the hardware platform (C / C ++, OpenVXgraphs, OpenCL *, etc.)
  • Seamless integration with system software stack and applications

In addition to these tools, Intel developed Intel MKL-DNN ‚Äî high-performance open-source APIs for deep learning software in conjunction with the Intel MKL library. MKL-DNN is available now, the release of the Intel Deep Learning SDK is scheduled for October, here you can sign up for a preview.
