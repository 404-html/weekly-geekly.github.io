Open source mail service Lavabit



Lavabit is an anonymous postal service ( dark mail ) that Edward Snowden used while in Sheremetyevo in the summer of 2013. As soon as the American intelligence services found out about this, they forced the owner of Lavabit, Ladar Lewison, to install monitoring equipment on the server. The FBI then discovered that the mail was encrypted and demanded that the owner give out the TLS secret keys. He refused. The trial began, the results of which Levison chose to close Lavabit and destroy the encryption keys.

Ladar Levison has already achieved the declassification of some documents on the case, and now he has decided to open the source code of the Lavabit mail service .

In the Github repository, the source code for the magma mail daemon, which supports server- side encryption, SMTP, POP, IMAP, and HTTP (with the JSON-RPC interface for the webmail client, it is also there).

The sources of the libdime library for multi-level encryption are also published Dark Internet Mail Environment (DIME, see protocol specifications ).
