DevConf :: BackEnd already this week on June 17th on Saturday, the program is formed

We invite you to participate in DevConf :: BackEnd this Saturday, in the center of attention: PHP 7.1 , weakness of the network API in the Linux kernel , chat bots, live video broadcasts, event processing systems and Haskell .

The conference is opened by an interesting talk by Valentin Bartenev, a member of the Nginx development team. It will scold Linux - more precisely, the network kernel API for userland-based applications, and complain about the hard, multi-threaded multiplexing life.

Alexander Makarov, one of the authors of the Yii framework, will talk about where we are going with PHP 7.1. Artem Dekhtyar and Pavel Stepanets will tell PHP people about the unequal struggle against legacy. Ivan Matveyev will convince you to write in PHP something that you can also read. And for those who love fashionable, stylish and strange, Dmitry Zuykov will tell you about Haskell web development.

Chatbots for the telegram remain a fashionable topic. Last year, the platform for bots was presented on Microsoft and Ruby stacks, and this year they decided to catch up with PHP , Python and Rust . Vladimir Yuldashev and Anton Shramko will teach us to use the Telegram API completely and correctly.

Heavy artillery this year are Efim Pyshnograev from Yandex, Alexander Serbul from 1C-Bitrix, and Alexey Akulovich from Vkontaktik. We will hear fascinating and instructive stories about the processing of billions of rubles, that is, events in AppMetrica , about live video broadcasts, and the taming of the Amazon by a cluster of bitrixes.

DevConf'17 conference partners: Alpha Laboratory, MediaSoft, Selectel, Reg, Ru, Lamoda, YiiConf, JoomlaDay, Postgres Professional, PGDAy, Lua in Moscow, ElixirLangMoscow, Frontend Weekend, PHPClub.ru, Meetup.com

See you at the developer community conference!
