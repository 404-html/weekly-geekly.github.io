Alda - music programming language


Quite recently, the release of the Alda programming language appeared on GitHub.

What is Alda?
The goal pursued by the creator of the language is the foundation of a new powerful and flexible programming language that will allow you to create musical compositions of various genres in a text editor.

Perhaps we are on the verge of opening a new profession (or movement) of a programmer - a musician. We do not know when such a profession will become popular, but we know for sure that we are able to test programming skills, Alda is no exception, we are already composing tasks for creating music, they will be evaluated by comparison with a given set of tracks.

We'll tell you more later, but for now you can see the documentation for Alda.

By the way, if you already have experience in creating music on Alda - please share your success!
