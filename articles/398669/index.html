First Rumor: OLED Display and Touch ID in the New Macbook Pro



Attentive comrades from MacRumors noticed in Apple's macOS Sierra 10.12.1 release yesterday hidden Apple Pay images randomly showing off a new Macbook Pro, which is expected to be released in the evening (Moscow time) on Thursday.

In the image we see an elongated touchscreen (the Cancel button on the left) display (the publication suggested it is an OLED) and a darkened area Touch ID from the right edge. In TechCrunch, we got the name Magic Toolbar from memory . You might think that the company intends to integrate a biometric payment tool not only into its own products, such as iTunes and the App Store, but also further into the web and third-party applications.

Also, due to this constructional decision, the speakers can be moved to the sides of the keyboard.

Other rumors already told in the run-up to the official announcement include the possible use of Intel Skylake processors ( others contradict them, integration of support for ARM Hurricane) and integration of USB Type-C port and connector, in particular, as a main charger, as well as the company's work on E-Ink keyboards for future devices.



On Thursday, October 27, text broadcast of the Apple event will start on Hiktaimes, beginning at 20:00 Moscow time (GMT +3).
