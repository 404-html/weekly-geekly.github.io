Hacker put up for sale 640 thousand stolen user accounts PlayStation



The user of one of the hacker forums in the darknet under the nickname SunTzu583 put up for sale 640,000 PlayStation user accounts. The data dump contains email addresses and unencrypted user passwords. The cost of "goods" is only $ 35.71 (0.0292 Bitcoin), the source of the leak is unknown.

Previously, the same seller put up for sale the data of 1 million users of Gmail and Yahoo. According to the seller, the PlayStation users data archive was not really stolen from the Sony network, but it contains unique data. SunTzu583 noted that data may open access to other user accounts, since many use the same usernames and passwords to access different sites, but ‚Äúthese are primarily PlayStation accounts‚Äù.

A few months ago, PlayStation users complained about hacking accounts and stealing money from accounts. Sony has denied the possibility of hacking its servers, but did not rule out the possibility of entering third-party accounts using data stolen from other leaks.

It is not the first time that gaming platforms have become the target of hackers - recently, the Clash of Clans game forum, as well as Epic Games and the ESEA community, have been hacked .
