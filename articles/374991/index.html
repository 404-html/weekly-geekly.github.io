Chromixium: Ubuntu ... that looks like Chrome OS



The cloud operating system Chrome OS is all good, but many users lack native desktop applications. Previously, there was no special choice, but now the Chromixium operating system has appeared , which is a customized version of Ubuntu Linux. Customization is to change the interface and some other parameters of the OS. In appearance, this is Chrome OS, in functionality - Ubuntu, with all the attendant features.

At the same time, under this OS, you can run any Chrome web application, synchronize services, browsing history in the browser and all other parameters with your Google account. As for the desktop, the start menu and the taskbar - all this is stylized under Chrome OS.

But we remember that inside is Ubuntu, with the ability to run any related desktop applications, including LibreOffice, GIMP, Firefox, Steam and other software. The taskbar contains shortcuts to YouTube, Gmail, and Google Drive, plus there is an opportunity to open a menu with Chrome OS apps.



Right-clicking on the desktop allows you to access other applications, change the wallpaper, access the settings, run the command line. There is also a Synaptic package installer, it is possible to install applications using apt-get.
