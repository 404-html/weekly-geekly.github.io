Microsoft and Arduino will become partners, and Windows 10 - the first officially licensed OS for Arduino



Yesterday, in the Microsoft blog and today - in the Arduino blog there was news about the agreement concluded by the two companies. This will allow you to create software for Windows 10, which will be closely related to iron projects running on the Arduino, and will give Arduino projects enhanced capabilities.

For example, the developer will be able to make a surveillance camera using the Arduino, which controls its turns and tilts. And under Windows, using the Universal Windows Platform, create a convenient interface for it, connect it to the cloud system, process incoming pictures in order to recognize movements, faces and / or voices.

The initial version of the system will contain two key components:

- Windows Virtual Shield for Arduino - access to the capabilities of devices compatible with Windows 10, wirelessly from the Arduino board. The same Lumia 530 contains so many sensors and other features that it would be necessary to spend $ 200 to hang them on the Arduino as shields. Imagine an Arduino project that has access to GPS, access to websites, a touchscreen display, speech technology, and so on. ( sample project )

- Windows Remote Arduino - control of the Arduino board from any of the devices running Windows 10. All available device functions are combined with the ability of the Arduino to manage things from the physical world ( example of a project ).
