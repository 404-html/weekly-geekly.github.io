Most companies provide this service by special requests or provide an opportunity to get acquainted with the systems in demos. In addition to the opportunity to see the work of the platforms, they have features that distinguish them in a large stream of emerging IOT services.

eZhing




The platform allows:


  • create your own interactive site plans (apartments, houses, greenhouses, or even large workshops of factories and factories)
  • plot on the layout of objects from the IOT sphere (sensors, signaling devices, ...)
  • link created plans to locations on a geographical map

Using the platform, you can provide geo-monitoring of objects in real time.



Thingsboard




Thingsboard is an open source platform. She has good documentation and examples of use cases.


The platform allows:


  • quickly develop, maintain and scale IoT projects
  • integrate with other IoT data processing services (AWS IoT, Apache Spark, SigFox, IBM Watson IoT)
  • relay device data to other systems
  • connect devices using standard IoT - MQTT, CoAP and HTTP protocols
  • create dashboards for data visualization

The platform has many other useful and original features.



LEVEREGE




The platform is designed to quickly build prototypes of IoT systems and effectively refine solutions to meet the needs of customers.


The platform allows:


  • create ioT object simulators
  • work with different IoT protocols
  • create and flexibly customize application interfaces
  • customize the business logic of the application

Using the platform, you can quickly provide end users with the opportunity to get acquainted with the options for solutions and get feedback from them
