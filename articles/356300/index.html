On Windows 10, you can run applications developed for Android and iOS



At the Build 2015 conference, Microsoft Vice President of Operating Systems Terry Myerson said the following:

"Windows Studio will allow developers to use any Java and C ++ code from Android applications in applications for Windows 10," after which he added that the programs developed in Xcode, used to create applications on iOS, will also work on Windows 10.

That is, any application created for Android or iOS can be adapted for Windows 10 with a few clicks. This opens up tremendous prospects for the platform, which has so far been criticized for its poor store: it will be possible to significantly expand the range of the current 300,000 programs with 1.3 million Android and 1.2 million iOS applications. They will have access to all the features of the Windows platform, including Microsoft Surface tablets, Lumia smartphones, Xbox One console, Windows 10 for PCs and laptops, as well as upcoming Microsoft HoloLens .
