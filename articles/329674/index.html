Solid12 - Ultimate IEM Platform Version for Free PostgreSQL

The Ultimate 2C limited-free IEM system for small businesses will now be delivered exclusively on the Solid12 platform.

Customers of industrial Ultimate IEM solutions will have the opportunity to choose between the enterprise version of the Ultimate Solid platform, aggregated with Oracle Database 12c EE, and the light version of Solid12 for PostgreSQL.

Migrating to a free DBMS has reduced the cost of the Ultimate Cloud cloud service : from 129 to 99 rubles per user per day.

The functionality of the IEM-platform Solid12 is somewhat reduced compared to the enterprise version (in proportion to the simpler functionality of a free DBMS compared to the enterprise market leader), but not to the detriment of operational properties for medium and small businesses.

Here is a complete list of the differences between Solid12 and Ultimate Solid:

  • Predictive access is not supported.
  • Temporary types (DateTime, timestamp) are not translated according to the user's time zones. All data is displayed in the same time zone.
  • There is no task to automatically remove blocking sessions.
  • It is forbidden to read several DataReaders in parallel on one connection with the database (MARS - Multiple Active Result Sets).
  • Any database error prevents the execution of further commands and committing the current transaction.
  • There is no information about the user currently blocking the branch of the business logic space.

We will develop the Ultimate Solid12 platform in parallel with the enterprise version, and will try to minimize the differences in functionality.
