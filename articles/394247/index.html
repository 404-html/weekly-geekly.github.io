Google released Spaces messenger, integrated with YouTube, Photos and Chrome



Today, the search giant announced the release of a new social application Spaces. True, you can call it a messenger only conditionally: although the program is intended for communication, in fact it is a whole set of Google applications seamlessly combined into one social platform. The company is positioning Spaces as a tool for communication in small groups of users on any topic that can be found on the Internet using the built-in Chrome browser. In this case, it will be possible to share YouTube videos and photos from Google Photos, using them as an excuse to create new discussions without having to go to each application separately.

On the official website of the Spaces project, it is not yet possible to download and view the messenger, since its release should take place today - therefore, it can only be judged by the screenshots listed there. Apparently, Google offers the following scheme of work with the application: on the start screen, very similar to the mobile screen of Chrome, you can use the search to find interesting material, then using a special button you can either create a discussion and invite friends to it, or join the conversation already in progress.
