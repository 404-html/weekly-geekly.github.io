Top 3 Mobile Software Security Testing Innovators

On December 19, 2018, IDC analytical agency published a report from IDC Innovators: Mobile App Security Testing, 2018 , in which it presented three innovators from 2018, offering mobile application security testing solutions (Mobile App Security Testing - MAST). These companies are High-Tech Bridge, Kryptowire and NowSecure.


Mobile software developers (as well as any other software) are not immune from errors, so numerous codes of mobile applications often reveal holes that make them vulnerable to user data theft or unauthorized use. Manual verification of software code is too time-consuming for many programmers and development companies, and, moreover, is also subject to human error.




Innovations in the field of testing the security of mobile and embedded applications today are artificial intelligence (AI) and various tools to automate the code testing process. All this allows even at the stage of application development to reduce the risks of their hacking and make them more secure, and also greatly reduce the time to check the code and, of course, the overall development time.

Innovation Companies 2018


Traditionally, IDC Innovators reports are compiled for different sectors of the market, while for each of them a large number of companies (but not all) are considered with annual revenues of less than $ 100 million at the time of selection for the survey. In the process of preparing these reports, IDC analysts study companies offering innovative technologies, an innovative approach to the existing problem and / or a new business model in a particular market. Such companies in the report IDC Innovators: Mobile App Security Testing, 2018 - High-Tech Bridge, Kryptowire and NowSecure.


High-Tech Bridge uses machine learning and artificial intelligence to check the security and privacy of mobile applications. This solution uses the data obtained at the stages of previous application tests and real-life use cases, as well as manual testing in difficult cases, in order to reduce the number of false positives.


Kryptowire offers technologies that test software for mobile devices (including firmware) and embedded software for IoT devices for compliance with international security and privacy standards. This allows you to effectively increase the level of software security, eliminating the detected vulnerabilities at the programming stage.


NowSecure offers a solution for automatically testing the security of a mobile application - the system itself embeds the real attack code into the file of the application under test for its dynamic and behavioral testing. Due to the frequent update of AI in NowSecure solutions, their use can effectively replace testers.
